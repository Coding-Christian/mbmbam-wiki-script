for creating templates to upload to the Fandom wiki

# Requirements
- WSL 2 `ubuntu 18.04`
- Node `16.1.0`

# Dependencies
- `axios`

# Usage
```shell
//in WSL terminal
git clone https://github.com/Coding-Christian/mbmbam-wiki-script.git
cd mbmbam-wiki-script
npm install

node main.js [episode_number] | clip.exe
```
